# Security Policy

## Supported Versions

Only the latest release of VisaKit is actively supported with security updates.

| Version | Supported |
| ------- | --------- |
| < 0.3.0 | No        |
| 0.3.x   | Yes       |

## Our Security Guarantees

VisaKit is a local-first browser extension. We guarantee the following security policies:
1.  **Local Storage only**: Your passport numbers, contact credentials, name details, and dates are saved strictly locally inside Chrome's secure extension storage.
2.  **No Server Communications**: We host no backend services, databases, or API sync networks. Your profile never travels across the internet.
3.  **No Telemetry**: We do not load error monitoring frameworks (like Sentry), tracking cookies, or website traffic analytics (like Google Analytics).

## Reporting a Vulnerability

If you discover a security vulnerability, please do NOT file a public issue on GitHub. Instead, email a report to masudul.dev@gmail.com. We aim to respond within 48 hours and fix issues promptly.
