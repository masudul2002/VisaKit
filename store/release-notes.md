# VisaKit v1.0.0-rc1 Release Notes

*   **RC1 Candidate**: Complete monorepo scaffold ready for Chrome Web Store distribution.
*   **Decoupled Matching Engine**: Maps portal fields dynamically using fuzzy normalizers and priority attribute lists.
*   **Resilient Loop**: Recovers gracefully if single fields fail.
*   **Integrated Options & Settings**: Sleek sidebar design linking profiles dashboards, backups import/export, and theme changes.
