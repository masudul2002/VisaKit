# Frequently Asked Questions

### Q1: Is my data safe with VisaKit?
**A**: Yes. VisaKit stores all profile details locally in your browser's Chrome storage. The extension does not use servers, cloud databases, or telemetry tools, keeping all data fully offline.

### Q2: Does it bypass CAPTCHAs or submit the form for me?
**A**: No. VisaKit fills input text, selectors, and options, but does not interact with CAPTCHAs, resolve validation challenges, or click submit buttons.

### Q3: How do I backup my saved profiles?
**A**: Navigate to the Settings tab in the Profile Manager dashboard and click "Export" to download a versioned JSON backup file.
