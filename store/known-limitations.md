# Known Limitations & Scope Boundaries

*   **No CAPTCHA Solvers**: Users must resolve security CAPTCHAs manually. VisaKit does not contain OCR or automation tools for security bypasses.
*   **Form Submission**: The user must review all details and click "Submit" manually. No automated submissions are supported.
*   **Browsers Support**: Limited to Chromium-compatible engines (Brave, Chrome, Edge). Firefox and Safari are not currently supported.
