# Chrome Web Store Release Guide

Steps for submitting the zip file to the Chrome Developer Dashboard:

1.  **Dashboard Access**: Navigate to `https://chrome.google.com/webstore/devconsole` and log into your developer account.
2.  **Create Item**: Click the **Add new item** button in the dashboard top-right.
3.  **Upload ZIP**: Upload [visakit-v1.0.0.zip](file:///c:/Users/Masudul/VS%20Code/Project/VisaKit/extension/visakit-v1.0.0.zip).
4.  **Populate Store Listing**:
    *   Copy descriptions from [store-listing.md](file:///c:/Users/Masudul/VS%20Code/Project/VisaKit/store/store-listing.md).
    *   Upload graphic assets (icon, screenshots, promotional banners) from [assets/store/](file:///c:/Users/Masudul/VS%20Code/Project/VisaKit/assets/store/).
5.  **Privacy Settings**: Declare offline storage features (using [privacy-summary.md](file:///c:/Users/Masudul/VS%20Code/Project/VisaKit/store/privacy-summary.md)).
6.  **Submit**: Click **Submit for review**. Review times range from 24 hours to 3 days.
