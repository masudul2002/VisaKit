# Chrome Web Store Listing Visual Assets

Place the required brand and promotional artwork in this folder before publishing.

## 🎨 Visual Assets Dimension Check

1.  **Extension Icon (128x128px PNG)**:
    *   This is the primary logo displayed in the store listing details page and extension settings.
    *   Path: `extension/public/icon-128.png` (already generated!).
2.  **Small Tile (440x280px PNG)**:
    *   Displayed in the Chrome Web Store search results page and category overview grids.
3.  **Large Tile (920x680px PNG)**:
    *   Used in general feature lists and promotional banner slides.
4.  **Promotional Image (1400x560px PNG)**:
    *   Used on the store home carousel if selected for a featured spot.
5.  **Screenshots (Min 1280x800px or 640x400px)**:
    *   Add at least 2-3 screenshots showcasing:
        *   `screenshot_dashboard.png`: The popup profile status and quick actions layout.
        *   `screenshot_options.png`: The CRUD options profiles list and validation fields.
        *   `screenshot_report.png`: The autofill report modal after form execution.
